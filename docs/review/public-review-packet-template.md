# Public Review Packet Template

Use this template for future public review packets.

## Packet ID

Stable identifier, for example `permea_core_public_review_packet_v0`.

## Title

Human-readable packet title.

## Purpose

Why this public review packet exists.

## Intended Reviewer

Researcher, reviewer, collaborator, investor, partner, maintainer, or public technical reader.

## Review Objective

What the reviewer should be able to answer after completing the packet.

## Required Reading Path

README -> QUICKSTART -> REVIEW PACKET -> EVIDENCE -> BENCHMARKS -> DATASETS -> RESEARCH -> CLAIMS -> VALIDATION

## Required Commands

Commands required for review, starting with:

```bash
python3 scripts/permea_review.py
```

Evidence assets and limitations linked by the packet.

## Benchmark Summary

Benchmark registry, lifecycle, and limitation links.

## Dataset Summary

Dataset registry, lifecycle, provenance, and limitation links.

## Research Package Summary

Research package registry, assembly, and limitation links.

## Claim Boundary Summary

Supported claim categories, unsupported claims, and claim registry links.

## Validation Summary

Validation and reproducibility commands, expected outputs, and known gaps.

- no solved-delivery claim
- no SOTA performance claim

- no expression improvement claim

Known review, evidence, benchmark, dataset, research package, claim, validation, and reproducibility limitations.

## Version
