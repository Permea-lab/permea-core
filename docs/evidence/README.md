# Permea Evidence Layer

The Evidence Layer connects Permea Core public claims to generated artifacts, reports, validation surfaces, decision records, and limitations.

It is a public review layer. It does not add scientific evidence, execute acquisition, run models, or expand claims.

## Status Labels

- Implemented: merged public documentation, generated artifact, or command exists.
- Prepared: public structure exists but future work is expected.
- Historical: retained for context, not current direction by itself.
- Public-Safe: reviewed for bounded public claims and non-claims.
- Needs Refresh: should be updated before it is used as current evidence.

## Required Non-Claims

- no dataset downloaded
- no acquisition executed
- no redistribution rights confirmed
- no wet-lab validation by Permea
- no clinical efficacy claim
- no model performance claim
- no SOTA claim
- no solved-delivery claim
