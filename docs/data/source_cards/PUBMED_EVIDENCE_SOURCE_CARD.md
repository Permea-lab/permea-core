# PubMed Evidence Source Card

| Field | Value |
| --- | --- |
| source_id | pubmed |
| source_name | PubMed / NCBI literature metadata |
| public URL or citation | https://pubmed.ncbi.nlm.nih.gov/ and NCBI E-utilities documentation, to verify |
| delivery axis | literature evidence graph |
| data expected | bibliographic metadata, PMID, DOI, title, abstract where permitted, publication type, links to full text where available |
| label type | evidence source pointer; extracted claims require separate evidence-card review |
| acquisition mode | manual citation review first; later metadata harvester only under NCBI usage rules |
| license/access status | metadata access is public; abstract/full-text reuse requires copyright and source review |
| redistribution risk | no redistribution of article text unless record-level rights permit it |
| provenance fields to preserve | PMID, DOI, title, journal, publication date, authors, source URL, access date, evidence extraction note, reviewer |
| expected benchmark use | evidence cards for BBB, CPP, localization, and benchmark source claims |
| limitations | PubMed is an index and citation source; it does not itself grant reuse rights for all linked content |
| immediate next action | define citation metadata fields and copyright-safe extraction policy before automated use |

## Notes

PubMed should first support citation provenance and source discovery. Evidence
cards should keep extracted claims narrow and preserve PMID/DOI links and review
status.
